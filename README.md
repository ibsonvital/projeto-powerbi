# 📊 Projeto Power BI - Análise de Vendas

Este projeto foi desenvolvido para análise de dados fictícios de vendas utilizando o Power BI.

## 🗂️ Base de Dados

- Arquivo: `vendas`


## 📈 Métricas analisadas

- Custos
- Impostos
- Total Vendido
- Lucro Líquido
- % Lucro

## 🧰 Ferramentas utilizadas

- Power BI
- Excel
